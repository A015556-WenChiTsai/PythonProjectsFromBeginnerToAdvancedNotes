# 剪刀石頭布 (Rock, Paper, Scissors) - Python 學習專案

歡迎來到 Python 學習專案！本專案透過經典遊戲「剪刀石頭布」，展示了從 **初學者腳本** 到 **進階模組化編程** 的思維轉變。

本目錄包含兩個版本的程式碼，分別代表不同的學習階段。

## 📂 檔案介紹

### 1. `3_rock_paper_scissors_v2.py` (基礎版)
這是適合初學者的版本。所有的邏輯都寫在一個主要的 `while` 迴圈中，程式碼由上而下執行，直觀易懂。
*   **特點**：包含計分功能（記錄玩家與電腦的勝場數）。
*   **重點**：學習基礎的流程控制與變數操作。

### 2. `3_rock_paper_scissors.py` (進階結構版)
這是更接近軟體工程師寫法的版本。將功能拆解為不同的 **函式 (Functions)**，讓程式碼更易於維護與閱讀。
*   **特點**：使用函式封裝邏輯，並包含輸入驗證。
*   **重點**：學習模組化設計、`random.choice` 以及程式進入點的概念。
