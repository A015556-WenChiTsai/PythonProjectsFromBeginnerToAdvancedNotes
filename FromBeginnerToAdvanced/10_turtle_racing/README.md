# 🐢 Python 烏龜賽跑遊戲 (Turtle Racing Game)

## 專案簡介

這是一個使用 Python 內建的 `turtle` 繪圖庫製作的圖形化賽跑模擬遊戲。使用者可以決定參賽的烏龜數量（2 到 10 隻），程式將隨機分配顏色並開始比賽。

這個專案旨在幫助初學者掌握 Python 的核心流程控制、函式定義、使用者輸入驗證以及基本的圖形化介面操作。

## 🚀 如何運行

1.  **確保環境:** 您只需要安裝標準的 Python 3 環境。`turtle`、`random` 和 `time` 都是內建庫。
2.  **執行檔案:** 在終端機中導航到專案目錄，並執行：

    ```bash
    python 10_turtle_racing.py
    ```

3.  **互動:** 程式啟動後，請在終端機中輸入您希望的參賽者數量 (2-10)。

## 程式碼結構解析

### 1. 參數設定

*   `WIDTH, HEIGHT`: 定義了遊戲視窗的大小。
*   `COLORS`: 包含所有可用的中文顏色名稱列表。
*   `COLOR_MAP`: **關鍵字典**，將中文顏色名稱映射到 `turtle` 庫能識別的英文顏色字串。

### 2. `get_number_of_racers()`

此函式負責獲取使用者輸入的參賽者數量。它使用一個 `while True` 迴圈來強制使用者輸入一個介於 2 到 10 之間的有效整數。

### 3. `create_turtles(colors)`

*   計算每隻烏龜的水平間距 (`spacingx`)，確保它們均勻分佈。
*   遍歷選定的顏色列表，為每個顏色創建一個 `turtle.Turtle` 物件。
*   使用 `COLOR_MAP` 獲取英文顏色，並設定烏龜的顏色和形狀。
*   使用 `racer.left(90)` 讓烏龜朝上，並使用 `racer.setpos()` 將它們定位在起跑線。

### 4. `race(colors)`

這是遊戲的核心執行部分：

*   進入無限迴圈，模擬比賽持續進行。
*   在每次迴圈中，遍歷所有烏龜，並讓它們隨機向前移動 1 到 19 個單位。
*   檢查每隻烏龜的 Y 座標。一旦 Y 座標達到或超過終點線 (`HEIGHT // 2 - 10`)，則宣告該烏龜獲勝，並返回其顏色。

### 5. 主程式流程

1.  呼叫 `get_number_of_racers()` 獲取參賽數量。
2.  呼叫 `init_turtle()` 初始化視窗。
3.  使用 `random.shuffle(COLORS)` 隨機打亂顏色列表，並選取所需數量 (`colors = COLORS[:racers]`)。
4.  呼叫 `race(colors)` 開始比賽並獲取贏家。
5.  列印結果並使用 `time.sleep(5)` 保持視窗開啟 5 秒，以便觀察結果。
