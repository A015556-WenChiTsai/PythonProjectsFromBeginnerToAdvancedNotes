# 🌲 Python 文字冒險遊戲 (Text Adventure Game)

這是一個適合 Python 初學者的文字冒險遊戲範例。透過簡單的互動式劇情，展示了 Python 基礎語法與程式邏輯控制的應用。

## 📖 專案簡介

玩家將扮演一名在迷霧森林邊緣醒來的角色，透過輸入數字選擇不同的路徑，決定自己的命運。遊戲包含多個結局，取決於玩家在關鍵時刻的決策。

## 🎯 學習目標

透過這個專案，你可以學習到以下 Python 核心概念：

- **函式 (Functions)**：利用 `def` 將遊戲場景模組化。
- **流程控制 (Control Flow)**：使用 `if-elif-else` 處理玩家的選擇。
- **輸入與輸出 (I/O)**：使用 `input()` 獲取輸入與 `print()` 顯示劇情。
- **參數傳遞**：如何在函式之間傳遞資訊（如結局訊息）。
- **程式進入點**：理解 `if __name__ == "__main__":` 的用途。

## 🚀 如何執行

1. 確保你的電腦已安裝 Python 3.x。
2. 下載 `4_adventure_game.py` 檔案。
3. 在終端機 (Terminal) 或命令提示字元 (CMD) 中執行以下指令：

```bash
python 4_adventure_game.py
