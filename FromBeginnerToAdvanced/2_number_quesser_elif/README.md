# 🔢 Number Guessing Game (數字猜猜樂)

這是一個適合 Python 初學者的互動式終端機小遊戲。程式會產生一個隨機數字，並引導使用者透過提示（太大或太小）來猜出正確答案。

## 🚀 功能特色

- **自定義範圍**：使用者可以自行決定要猜的數字最大範圍。
- **輸入驗證**：包含基本的防呆機制，確保輸入的是有效數字。
- **即時回饋**：每次猜測後，程式會提示「太大」或「太小」。
- **次數統計**：遊戲結束後，會顯示你總共猜了幾次才答對。

## 🛠️ 技術重點 (Tech Stack)

這個專案展示了 Python 的基礎核心概念：
- `random` 模組應用
- `while` 迴圈與 `break`/`continue` 控制
- `if/elif/else` 條件判斷
- `input()` 處理與 `int()` 型別轉換
- `isdigit()` 字串驗證方法
