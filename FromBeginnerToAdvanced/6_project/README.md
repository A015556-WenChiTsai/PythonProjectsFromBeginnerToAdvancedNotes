# Python Beginner Projects Collection 🐍

這是一個包含三個 Python 基礎實作專案的集合。這些專案旨在練習 Python 的核心語法、流程控制、資料結構以及模組應用。

## 📂 專案列表

### 1. 多人骰子遊戲 (PIG Dice Game)
**檔案：** `6_project1.py`

這是一個經典的風險管理骰子遊戲。2 到 4 名玩家輪流擲骰子，累積回合分數。如果擲出 1，該回合分數歸零並換人；如果選擇停止，回合分數將加入總分。先達到 50 分者獲勝。

**學到的技術點：**
- **流程控制**：複雜的 `while` 與 `for` 巢狀迴圈應用。
- **輸入驗證**：確保使用者輸入有效的人數與指令。
- **列表操作**：使用 List Comprehension 初始化與追蹤多位玩家分數。
- **邏輯判斷**：實作遊戲規則與獲勝條件檢查。

### 2. 故事填詞產生器 (Mad Libs Generator)
**檔案：** `6_project2.py`
**依賴檔案：** `story.txt` (需包含 `<形容詞>`, `<名詞>` 等標籤的文本)

程式會讀取外部故事檔案，自動偵測並提取尖括號 `< >` 中的詞性標籤，提示使用者輸入對應的詞彙，最後生成並印出一個全新的有趣故事。

**學到的技術點：**
- **檔案讀寫 (I/O)**：使用 `with open` 安全地讀取文本。
- **資料結構**：
  - 使用 `set` (集合) 去除重複的替換詞。
  - 使用 `dict` (字典) 建立標籤與使用者輸入的映射關係。
- **字串處理**：使用 `enumerate` 遍歷字元並實作自定義的解析演算法 (Parsing Logic)。
- **字串替換**：使用 `replace` 方法組合最終結果。

---

### 3. 數學隨堂測驗 (Timed Math Quiz)
**檔案：** `6_project3.py`

一個自動生成隨機數學題目（加、減、乘）的測驗程式。系統會記錄答錯次數，並在完成 10 題後計算總花費時間。

**學到的技術點：**
- **標準函式庫**：
  - `random`：隨機選取運算符號與生成數字。
  - `time`：計算程式執行時間（時間戳記操作）。
- **動態執行**：使用 `eval()` 函數動態計算字串形式的數學表達式。
- **常數定義**：使用全大寫變數管理遊戲配置（如題目數量、數字範圍）。
- **錯誤處理**：使用迴圈強迫使用者直到答對為止。
