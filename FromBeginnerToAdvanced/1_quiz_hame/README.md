這是我在 **FromBeginnerToAdvanced** 學習路徑中的第一個練習專案。這是一個基於終端機（Console）的互動式問答遊戲，程式會詢問使用者問題，判斷對錯並計算總分。

## 📝 專案簡介
這個腳本展示了 Python 程式設計中最基礎且核心的概念，模擬了一個簡單的「輸入 -> 處理 -> 輸出」流程。

## 🚀 學習重點 (Key Learnings)

透過這個範例，我掌握了以下 Python 核心技術：

### 1. 輸入與輸出 (I/O)
- 使用 `print()` 將訊息顯示在螢幕上。
- 使用 `input()` 暫停程式並獲取使用者的鍵盤輸入。

### 2. 流程控制 (Control Flow)
- **條件判斷**：使用 `if` 和 `else` 來決定程式的走向（例如：答對加分，答錯顯示正確答案）。
- **程式終止**：使用 `quit()` 在特定條件下（如使用者不想玩）直接結束程式。
- **邏輯運算子**：使用 `==` (等於) 和 `!=` (不等於) 進行比較。

### 3. 字串處理與資料清洗
- **大小寫轉換**：使用 `.lower()` 將使用者輸入標準化為小寫。這解決了使用者輸入 "Yes", "YES", "yes" 都能被程式識別的問題，提升了使用者體驗。

### 4. 變數與運算
- **變數宣告**：建立 `score` 變數來追蹤分數。
- **累加運算**：使用 `score += 1` (即 `score = score + 1` 的簡寫) 來更新分數。

### 5. 資料型態轉換 (Type Casting)
- 學習到 Python 不允許直接將「整數」與「字串」相加。
- 使用 `str(score)` 將數字轉型為字串，以便與提示文字進行串接輸出。
