# Python Password Manager (密碼管理器) 🔐

這是一個適合 Python 初學者的密碼管理器專案。本專案展示了如何從簡單的「明文儲存」進階到使用業界標準的「加密儲存」，並包含隨機強密碼生成功能。

## 📚 專案內容

本專案包含兩個版本的實作，分別對應不同的學習階段：

1.  **基礎版 (`5_password_manager.py`)**：
    *   專注於程式邏輯、檔案讀寫與使用者互動。
    *   使用 `secrets` 模組生成高強度隨機密碼。
    *   **注意**：此版本將密碼以明文儲存，僅供學習邏輯使用，不建議用於真實敏感資料。

2.  **進階加密版 (`5_password_manager_v2.py`)**：
    *   引入 `cryptography` 第三方庫。
    *   實作 **Fernet (對稱加密)** 技術。
    *   確保儲存在硬碟上的密碼是加密後的亂碼，只有持有金鑰的人才能解讀。

## 🚀 你將學到的核心技術

透過研究這個專案，你將掌握以下 Python 技能：

*   **檔案操作 (File I/O)**：使用 `with open()` 進行文字檔 (`.txt`) 與二進位檔 (`.key`) 的讀寫與追加模式操作。
*   **資料加密 (Encryption)**：學習如何使用 `cryptography.fernet` 進行加解密，以及 `bytes` 與 `string` 之間的編碼轉換 (`encode`/`decode`)。
*   **字串處理**：使用 `split()`, `strip()` 解析資料結構。
*   **模組應用**：使用 `os`, `random`, `secrets`, `string` 等標準庫。
*   **流程控制**：實作 CLI (命令列介面) 選單與無窮迴圈邏輯。
